搬瓦工与Vultr的搭建指南

搬瓦工主机购买

搬瓦工的主机按照年进行收费，所以在付款时是一次性买一台主机的一年的价格，跟阿里云的模式一样。

Vultr主机购买

Vultr主机采用的是按照小时扣费，并非年付方案，类似于网吧上网。

搭建操作方法

使用Secure CRT工具连接成功后，粘贴下面的命令进行操作：

[root@vultr ~]# yum install -y wget && wget --no-check-certificate -O shadowsocks-libev_CN.sh https://raw.githubusercontent.com/uxh/shadowsocks_bash/master/shadowsocks-libev_CN.sh && bash shadowsocks-libev_CN.sh  // 中文版
[root@vultr ~]# yum install -y wget && wget --no-check-certificate -O shadowsocks-libev.sh https://raw.githubusercontent.com/uxh/shadowsocks_bash/master/shadowsocks-libev.sh && bash shadowsocks-libev.sh  // 英文版

接下来需要安装锐速TCP加速软件，由于系统自带内核版本太高无法安装锐速，需要进行降级，复制命令进行操作：

[root@vultr ~]# wget --no-check-certificate -O rskernel.sh https://raw.githubusercontent.com/uxh/shadowsocks_bash/master/rskernel.sh && bash rskernel.sh

回车后系统会自动下载脚本并执行更换内核命令，按图提示，我们可以看到当前系统为CentOS7，等待内核更换完毕后系统会自动重启并断开连接：

在使用的过程中如果需要修改相关的配置信息，请使用下面的命令：

附一、修改Shadowsocks的配置信息

如果你以后需要修改Shadowsocks的配置（比如密码、端口或者加密），可以运行下列命令：

  • 中文版：bash shadowsocks-libev_CN.sh
  • 英文版：bash shadowsocks-libev.sh
  • 然后选择第 8 项：修改Shadowsocks配置即可重新设置Shadowsocks的密码、端口以及加密方式。

附二、卸载Shadowsocks服务

如果你以后需要卸载Shadowsocks服务，可以运行下列命令：

  • 中文版：bash shadowsocks-libev_CN.sh
  • 英文版：bash shadowsocks-libev.sh
  • 然后选择第 2 项：卸载Shadowsocks服务即可从服务器中卸载掉Shadowsocks服务。

客户端连接工具

相关资源直接在互联网下载即可。安卓手机使用SSRR客户端、苹果手机使用superwingy、Mac电脑使用ShadowsocksX-NG、Windows电脑使用Shadowsocks。

验证是否成功，可以直接访问谷歌看是否能够正常访问，或者访问（whoer.net）查看网络信息。
